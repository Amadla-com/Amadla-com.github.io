Welcome to Amadla’s documentation!

Amadla (/ah-mah-dlah/) is a set of IaC tools to help deploy and manage various cloud environments, dedicated servers, and on-premise systems.

Amadla is a Zulu word meaning “power” or “strength”.

Attention

This project is under active development.
